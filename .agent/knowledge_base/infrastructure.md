# Project Infrastructure

## OS & Environment
- **OS:** Windows 10/11
- **Shell:** PowerShell 5.1+
- **Editor:** VS Code

## SDK & Tools
- **Runtime:** Node.js
- **Package Manager:** npm
- **Languages:** TypeScript
- **Main Libraries:** Telegraf, screenshot-desktop, Chrome DevTools Protocol (CDP)

## Development workflow
- `npm run compile`: Build TypeScript to JavaScript
- Launch VS Code with `--remote-debugging-port=9222` for CDP features

## Branding
- **Project Name:** Antigravity Telegram Control
- **Color Palette:** Sleek dark modes, glassmorphism, vibrant blue/purple gradients (planned)
- **Icons:** Custom logo and icon set (in-progress)
